Today's topics:

  1. Here's that animation and object project from Tuesday! Download, read it, and play around with it! Make more and less objects, mess with the random number generators, and see how the class works to create objects. Don't worry if you don't 100% understand it, we'll keep at it all semester. There are a ton of challenging concepts here!
  2. Work on planning for your projects and begin coding. Here are some pointers:
    • Discuss roles for each member of your team. Keep everyone involved and helping out! Here are a few suggestions:
      • Planner/organizer -- helps coordinate members of the group and maintains focus on specific tasks.
      • Lead programmer -- writes code and hosts the main repository for the group on Github. Add the rest of the team as collaborators.
      • Lead graphics/designer -- communicates with programmer to determine pixel size of various assets.
      • Support members to handle programming, ideas, research, and graphics components as necessary.
      • Use Collab Edit for working on the same file at the same time.
    • Break your game down into manageable chunks and figure out how to accomplish those small bits one at a time. This is an extremely important strategy in problem solving! For example, when I wrote Tetris, I approached the game (roughly) in the following sequence of steps:
      1. Determine what my goal was and create a general plan.
      2. Set up variables, constants, and basic functions and objects.
      3. Get a square block to appear on screen.
      4. Get that block to move down the screen.
      5. Add keyboard input.
      6. Handle collisions for the active block to prevent it from going off the edges and to make it inactive when it reaches the bottom.
      7. Add more block types and write rotation functions for each block type.
      8. Add scores, images, and a second canvas element to hold the upcoming block.
  3. Getting started on projects is the hardest part, especially since this is essentially your first program. So we'll want to use the internet and tutorials to ensure we're headed down a reasonable path. See the class resources page and course materials pages for a full list of these resources or ask me directly.
