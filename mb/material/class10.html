Class 10: Javascript graphics

Today, we'll use everything we've discussed so far to build a graphics application, then spend some time walking through a Phaser tutorial. Once we complete these tasks, we'll dive into our group projects, which we'll focus on for the remainder of the semester.


Today's topics:

  1. Work as a class on some basic graphics concepts with JS canvas, one of the coolest parts of Javascript. Canvas allows you to create complex animations and games in the browser.
    • Canvas is useful for demonstrating general concepts of graphics-based programming. All graphics programs contain an initialization routine, which loads assets and sets variables to determine game parameters. Graphics programs also contain an update function, which updates the game state on every game frame and re-draws all necessary components. The update function is called multiple times per second by a timer.
    • Check out this canvas demonstration program which illustrates these concepts as well as collision detection, which is critical for game design and in some animations.
    • I have a bunch of more complex examples of graphics and normal JS scripts on Github such as snake, minesweeper, and tetris.
  2. Since canvas can be somewhat difficult to use, many libraries have been created which do a lot of the work for you, allowing you to focus on design rather than building everything from scratch. Mission Bit has used a library called Phaser, which makes game development easier by simplifying collision detection, physics, code organization, and graphics. We'll walk through a Phaser tutorial together to familiarize ourselves with the library and discuss which projects it's most appropriate for.
